1. VISÃO GERAL
===============================================================================

O CUMA é um aplicativo desktop para Windows voltado a limpeza, organização e
conversão de arquivos usados em mangás, quadrinhos, PDFs escaneados e leitura em
e-readers.

Ele concentra quatro fluxos principais:

1. Limpar PDFs, removendo páginas vazias ou pouco úteis.
2. Exportar o resultado como PDF, PDF + CBZ, CBZ ou imagens.
3. Usar ferramentas auxiliares para extrair páginas e montar PDFs a partir de
   imagens, PDFs e arquivos compactados com imagens.
4. Converter arquivos prontos para formatos como PDF, EPUB, CBZ, imagens e XTCH,
   conforme o tipo de entrada detectado.

A interface é dividida em áreas laterais:

• Limpar: fluxo principal de limpeza, preparação visual e exportação.
• Ferramentas: utilidades auxiliares, como extrair páginas e criar PDF de imagens.
• Converter: conversão pura de formato, sem relimpar nem reaplicar filtros visuais.
• Resultados: lista processos concluídos e arquivos gerados.
• Registros: mostra mensagens internas, logs e diagnósticos.
• Configurações: aparência, idioma, desempenho, segurança, logs e preferências.
• Sobre: resumo do aplicativo, versão e acesso aos manuais.

Regra mais importante:
• Use Limpar para preparar ou modificar o conteúdo.
• Use Converter para transformar o arquivo já pronto em outro formato.
• Use Ferramentas quando precisar de ações auxiliares fora da fila principal.

3. LISTAS DE ARQUIVOS E ORDEM MANUAL
===============================================================================

As abas Limpar, Ferramentas e Converter usam listas de arquivos. A partir da
versão 1.100.5, essas listas podem ser reorganizadas manualmente.

Como mover com o mouse:
1. Clique no arquivo desejado dentro da lista.
2. Segure o clique.
3. Arraste a linha para cima ou para baixo.
4. Solte na posição desejada.

Como mover com botões:
• Selecione um arquivo na lista.
• Clique em "Mover pra cima ↑" para subir uma posição.
• Clique em "Mover pra baixo ↓" para descer uma posição.

Observações:
• A ordem visível da lista é a ordem usada no processamento.
• Em Ferramentas > Criar PDF de imagens, essa ordem define a sequência das páginas
  no PDF final.
• Em lotes grandes, revise a ordem antes de clicar em Processar.
• Use os botões de mover quando quiser um ajuste preciso de uma linha por vez.
• Use arrastar e soltar dentro da lista quando precisar reorganizar vários itens
  visualmente.

4. ABA LIMPAR
===============================================================================

A aba Limpar é o fluxo principal do programa para PDFs.

Área "Arraste PDFs ou pastas aqui"
Permite soltar PDFs ou pastas diretamente na fila. Quando tkinterdnd2 não estiver
instalado, o aplicativo continua funcionando, mas o arrastar-e-soltar externo
pode ficar indisponível. Nesse caso, use os botões de seleção.

Adicionar PDF(s)
Adiciona um ou mais PDFs à fila.

Adicionar pasta
Adiciona PDFs encontrados dentro de uma pasta. Quando a opção de incluir
subpastas estiver marcada, também procura PDFs nas subpastas.

Colar caminho
Permite colar manualmente caminhos de arquivos ou pastas.

Remover
Remove da fila os itens selecionados.

Limpar lista
Esvazia a fila da aba Limpar.

Mover pra cima ↑ / Mover pra baixo ↓
Reordena o arquivo selecionado dentro da fila.

Prévia Limpar
Mostra a prévia correta das opções de limpeza e preparação visual. Essa prévia
serve para conferir alterações antes do processamento.

Configurações do PDF
Abre as opções que realmente modificam ou preparam o conteúdo antes da exportação.
Nessa janela ficam as opções de perfil visual e preparação, como:
• Ordem de leitura Ocidental/Mangá.
• Presets E-ink.
• Remoção automática de margens.
• Remoção de rodapé ou número de página.
• Divisão de páginas duplas.
• Manter a página original ao dividir.
• Webtoon/imagens longas.
• Prévia Limpar.

Pasta de saída
Define onde os arquivos limpos serão gravados. Essa pasta é independente da pasta
do Converter.

Sufixo
Texto adicionado ao nome final. Exemplo: "capitulo01.pdf" com sufixo "_limpo"
vira "capitulo01_limpo.pdf".

Formato de exportação
Define o tipo de saída:
• PDF: cria apenas o PDF limpo.
• PDF + CBZ: cria o PDF limpo e um CBZ com as páginas mantidas.
• CBZ: cria apenas o CBZ.
• Imagens JPG: exporta as páginas mantidas como JPG.
• Imagens PNG: exporta as páginas mantidas como PNG.

Intervalo de páginas
Permite limitar o processamento. Exemplos:
• 1-10: processa da página 1 até a 10.
• 1-5, 8, 12-15: processa blocos separados.
• vazio: processa todas as páginas.

Abrir resultado ao concluir
Quando marcado, tenta abrir a pasta ou o arquivo final ao terminar.

Processar
Executa a limpeza conforme as opções configuradas.

Quando usar Limpar:
• Quando o PDF precisa passar por preparação visual.
• Quando você quer remover páginas vazias ou pouco úteis.
• Quando quer criar PDF/CBZ/imagens já limpos.
• Quando quer aplicar corte, preset E-ink, divisão de página dupla ou webtoon.

5. ABA FERRAMENTAS
===============================================================================

A aba Ferramentas reúne utilidades auxiliares que não dependem da fila principal
da aba Limpar.

Ela possui duas subabas principais:

-------------------------------------------------------------------------------
5.1 EXTRAIR PÁGINAS
-------------------------------------------------------------------------------

A função Extrair páginas trabalha com PDFs.

Adicionar PDF(s)
Adiciona PDFs à lista de extração.

Adicionar pasta
Adiciona PDFs encontrados em uma pasta.

Colar caminho
Permite colar caminhos manualmente.

Remover / Limpar
Remove itens selecionados ou limpa a lista inteira.

Mover pra cima ↑ / Mover pra baixo ↓
Reordena os PDFs da lista.

Arrastar dentro da lista
Também é possível clicar, segurar e mover um arquivo para outra posição.

Processar
Extrai as páginas dos PDFs como imagens, conforme as opções disponíveis.

Quando usar:
• Para transformar páginas de PDF em PNG/JPG.
• Para separar páginas antes de montar outro arquivo.
• Para diagnóstico quando um PDF específico precisa ser conferido página por página.

-------------------------------------------------------------------------------
5.2 CRIAR PDF DE IMAGENS
-------------------------------------------------------------------------------

A função Criar PDF de imagens monta um PDF único a partir da lista exibida.

Tipos aceitos:
• Imagens: JPG, JPEG, PNG, WEBP, BMP, TIF e TIFF.
• PDF.
• CBZ.
• ZIP.
• EPUB com imagens.
• CBR, RAR e 7Z quando o 7-Zip estiver instalado e disponível no sistema.

Como adicionar:
• Use Adicionar arquivo(s) para selecionar imagens, PDFs ou arquivos com imagens.
• Use Adicionar pasta para inserir arquivos encontrados em uma pasta.
• Use Colar caminho para informar caminhos manualmente.
• Também é possível arrastar arquivos/pastas para a área da ferramenta quando o
  arrastar-e-soltar estiver disponível.

Ordem:
• A ordem da lista é a ordem do PDF final.
• Use "Mover pra cima ↑", "Mover pra baixo ↓" ou arraste linhas na lista para
  ajustar a sequência.
• Em CBZ/ZIP/EPUB com imagens, as imagens internas são lidas em ordem natural.
• Em PDFs adicionados à lista, as páginas entram na posição daquele PDF na lista.

Como o PDF final é montado:
• Imagens comuns entram como páginas.
• PDFs entram página por página.
• CBZ, ZIP e EPUB com imagens têm suas imagens internas extraídas temporariamente
  e inseridas no PDF.
• CBR, RAR e 7Z dependem do 7-Zip para extração.
• A ferramenta respeita a ordem manual da lista antes de gerar o arquivo.

Limitações:
• EPUB textual puro não é renderizado nessa ferramenta.
• EPUB com imagens é aceito.
• Arquivos CBR/RAR/7Z exigem 7-Zip instalado e disponível no PATH do Windows.
• Arquivos protegidos por senha podem falhar.

Quando usar:
• Para juntar capas, imagens soltas e capítulos em um PDF.
• Para criar um PDF a partir de um CBZ/ZIP/EPUB de imagens.
• Para juntar PDFs e imagens em uma sequência única.
• Para reorganizar manualmente capítulos antes de gerar um PDF final.

6. ABA CONVERTER
===============================================================================

A aba Converter transforma arquivos prontos em outros formatos. Ela foi ajustada
para ser mais compacta visualmente e não exibir o grande espaço vazio entre os
botões e a lista.

Regra da aba Converter:
• Converter apenas converte formato.
• Converter não relimpa o arquivo.
• Converter não remove margens.
• Converter não divide páginas duplas.
• Converter não reaplica presets E-ink.
• Converter não remove rodapé/número de página.
• Converter não aplica webtoon/imagens longas.

Use Limpar primeiro quando precisar modificar o conteúdo. Depois use Converter
para gerar o formato final.

Área "Arraste PDFs, EPUBs ou pastas aqui"
Permite soltar arquivos ou pastas. O texto pode variar conforme a versão, mas a
função é adicionar arquivos à lista de conversão.

Adicionar arquivo(s)
Adiciona arquivos compatíveis à fila do Converter.

Adicionar pasta
Adiciona arquivos compatíveis encontrados em uma pasta.

Colar caminho
Permite colar caminhos manualmente.

Remover
Remove os itens selecionados.

Limpar
Esvazia a lista do Converter.

Prévia
Mostra a prévia da entrada real que será enviada ao conversor, sem encaixar em
canvas branco de dispositivo nas conversões puras.

Funções por tipo
Mostra ou atualiza as conversões disponíveis conforme o tipo de arquivo selecionado.

Mover pra cima ↑ / Mover pra baixo ↓
Reordena os arquivos da fila do Converter.

Arrastar dentro da lista
Permite clicar, segurar e mover um arquivo para outra posição.

Dispositivo
Seleciona um perfil de dispositivo. O perfil é importante principalmente para
saídas que exigem dimensões específicas, como XTCH.

Aplicar perfil
Aplica o perfil escolhido às opções de conversão compatíveis.

Editor de perfis
Permite criar ou ajustar perfis de dispositivo.

Pasta de saída
Define onde o Converter salvará seus arquivos. Essa pasta é separada da pasta da
aba Limpar.

Metadados EPUB
Aparece somente quando o arquivo selecionado é EPUB ou quando alguma conversão de
saída para EPUB está marcada.

Conversões compatíveis por tipo:
• PDF pode liberar PDF → EPUB, PDF → XTCH, PDF → CBZ e PDF → imagens.
• EPUB pode liberar EPUB → XTCH e EPUB → PDF.
• CBZ/ZIP/CBR/RAR/7Z podem liberar compactado → EPUB, compactado → XTCH e
  compactado → PDF.
• Imagens podem liberar imagem → PDF, imagem → EPUB e imagem → XTCH.

Dependências:
• CBZ e ZIP funcionam nativamente.
• CBR, RAR e 7Z dependem do 7-Zip instalado no Windows.
• XTCH usa dimensões fixas de dispositivo por exigência do formato.

Quando usar:
• Para converter um arquivo que já está pronto.
• Para gerar EPUB de imagens, CBZ, imagens ou XTCH.
• Para adaptar arquivos a dispositivos Kindle, Kobo, reMarkable ou perfis
  personalizados.
• Para manter a preparação visual separada da conversão final.

14. LIMITAÇÕES CONHECIDAS
===============================================================================

• EPUB textual puro não é renderizado em Ferramentas > Criar PDF de imagens.
• EPUB com imagens é aceito nessa ferramenta.
• CBR/RAR/7Z dependem do 7-Zip instalado e disponível no PATH.
• tkinterdnd2 é opcional; sem ele, o arrastar-e-soltar externo pode não funcionar.
• Arquivos protegidos por senha podem falhar.

• A detecção de páginas vazias é visual/heurística; revise o PDF de removidas
  antes de apagar originais.
• XTCH exige dimensões fixas de dispositivo, então o perfil escolhido importa.

15. FLUXOS RECOMENDADOS
===============================================================================

Fluxo seguro para limpeza:
1. Abra o CUMA.
2. Vá para Limpar.
3. Adicione um PDF de teste.
4. Ajuste Configurações do PDF se necessário.
5. Use Prévia Limpar.
6. Processe.
7. Confira o resultado antes de rodar lotes grandes.

Fluxo para criar PDF a partir de imagens/PDF/CBZ:
1. Vá para Ferramentas.
2. Abra Criar PDF de imagens.
3. Adicione imagens, PDFs, CBZ, ZIP ou EPUB com imagens.
4. Reordene a lista com arraste ou com "Mover pra cima ↑" e "Mover pra baixo ↓".
5. Processe e confira o PDF final.

Fluxo para converter arquivo pronto:
1. Prepare o arquivo em Limpar, se ele precisar de ajustes visuais.
2. Vá para Converter.
3. Adicione o arquivo pronto.
4. Escolha dispositivo e formato de saída quando aplicável.
5. Confira as conversões compatíveis.
6. Processe.
7. Teste o resultado no leitor ou dispositivo real.

Fluxo para diagnóstico:
1. Abra Registros.
2. Copie as mensagens relevantes.
3. Verifique CUMA.log e erro.txt, se existirem.
4. Execute python cuma.py --debug-env quando precisar gerar diagnóstico completo.
